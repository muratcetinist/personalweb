# muratcetin.com - Personal Landing Page

## Project Overview
Personal landing page for Murat Çetin at muratcetin.com. Minimalist, dark mode design. Subdomains will host separate projects but won't be linked from this landing page.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel (recommended)

## Design Principles
- Minimalist and clean
- Dark mode by default
- Easy on the eyes
- No navigation to subdomains

## Commands
- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run lint` — Run linter

## Project Structure
```
src/
  app/          — Next.js App Router pages & layouts
  components/   — Reusable UI components
  styles/       — Global styles
public/         — Static assets
```
